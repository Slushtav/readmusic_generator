# 🎵 ReadMusic Generator

Generate a random music card for your GitHub README using a free music API.

<img src="https://img.shields.io/badge/Node.js-18+-339933?style=for-the-badge&logo=node.js&logoColor=white"/>
<img src="https://img.shields.io/badge/Vercel-Deployed-black?style=for-the-badge&logo=vercel"/>
<img src="https://img.shields.io/badge/API-Deezer-FF0066?style=for-the-badge"/>

*A simple project that generates a random song card as an SVG image for GitHub README.*

</div>

---

## ✨ Features

- 🎵 Random song every refresh
- 🎨 Beautiful SVG music card
- 💿 Album information
- 👤 Artist information
- ⚡ Serverless Function (Vercel)
- 🆓 100% Free (Deezer API)
- 📖 Easy to customize
- 🚀 Ready for GitHub README

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Slushtav/readmusic_generator.git
```

Go to the project

```bash
cd readmusic_generator
```

Install dependencies

```bash
npm install
```

Run locally

```bash
vercel dev
```

---

## 🚀 Deployment

Deploy with Vercel

```bash
vercel
```

or connect the repository directly from the Vercel Dashboard.
